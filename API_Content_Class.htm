Content Class

The Content class is the repository for all HTML produced by other classes. Content is stored in this object in the properties which are created by the add method. Common properties used throughout the system are $premain, $main and $postmain. After all classes are loaded and have had a chance to write to the $content object, it outputs the strings contains in its properties to the browser.

The global $content object is added to the global $vce object during site initiation, so make sure to call “global $vce” and then use then use $vce->content to call methods from the global $content object.

The Content class is found in class.content.php.

Properties
Name Type Description
$premain String Holds the content that by default will be written first to the content area of the final rendered page
$main String Holds the content that by default will be written after the premain content to final rendered page
$postmain String Holds the content that by default will be written after the main content to final rendered page
$admin String Holds admin content
javascript String HTML script content to add javascript files to the page in the normal header spot
javascript_footer String HTML script content to add javascript files to the page in the footer position

Note: these properties are not created during construct, but are the common values used by many components to add content to the $content object.

Methods
Name/Syntax Description
add ($property_name = 'main', $new_content = null, $prepend = false) Add content to the designated property of $content
menu ($title, $args = array()) Creates menu when called. Echos the final, formatted menu string so that it can be inserted directly into an HTML page context. Note that menu items are checked by role to ensure the viewer should have access to them.
output ($items = null) Either output current contents of $content, or values of $items if specified
Method Details
Add

Add content to the designated property of $content. Content will be rendered in the page area designated by the property. Content should be properly formatted for the PHP echo statement, to be rendered in HTML.

add ($property_name = 'main', $new_content = null, $prepend = false)

Parameters

$property_name String Designates to which $content property the content should be added, defaults to main. Values of ‘premain’, ‘main’ and ‘postmain’ are the most common, and by default these are the values that are written to the page by the output function.
$new_content String The content you will be adding. Should be formatted for HTML output
$prepend Bool When set to true, will prepend the content to the designated property. This is set to false by default, so in most cases content will be appended

Return

No return value

Example

$vce->content->add('main', '<br>Hiya!<br>');

Menu

Creates menu when called. Pulls the menu JSON object from the database and echos the final, formatted menu string so that it can be inserted directly into an HTML page context. Note that menu items are checked by role to ensure the viewer should have access to them.

menu($title, $args = array())

Parameters

$title String Title of the menu
$args Array Other args used by function, such as class and separator

Return

No return value, echo the string representing the menu

Example

//example of menu in a theme.php page, inserted into page HTML
<?php $vce->content->menu('main'); ?>

Output

Echos the current contents of the $content object. By default will concatenate premain, main and postmain and then echo to the screen. If $items are specified, output will look for properties matching $items array values, and output content in that order. Output is called in the theme files to insert dynamic content into the formatted page.

output($items = null)

Parameters

$items array of strings If set, this function will output all the properties designated in the array, in the order they appear in the array

Return

No return value

Example

//will concatenate admin, premain, main and postmain and then echo the result

$vce->content->output(array('admin', 'premain', 'main', 'postmain'));
